# System Monitor

A small Python script that shows your CPU and memory usage in real time. It updates every second and shows numbers along with small visual bars for an easy glance.

## Features
- Real-time CPU and memory stats
- Small visual bars for CPU and memory
- Shows current timestamp
- Works on Windows, macOS, and Linux

## Requirements
- Python 3.x
- `psutil` library

Install `psutil` with:

```bash
pip install psutil
```

## How to Use

1. Clone or download this repo.
2. Open a terminal in the project folder.
3. Run:
```bash
python monitor.py
```
4. Press Ctrl + C to stop.
